Panti Nugroho Mobile
Penjelasan Singkat

Panti Nugroho Mobile merupakan aplikasi mobile berbasis android, fungsi utama aplikasi ini adalah untuk melakukan pendaftaran secara online.
Baik pasien lama maupun pasien baru dapat menggunakan aplikasi ini untuk mendaftar ke poliklinik, selain itu juga dapat mendaftarkan anggota keluarga pemilik akun jika ingin melakukan pemeriksaan di poliklinik RSUD Panti Nugroho.

Menu Pada Aplikasi Panti Nugroho Mobile :

  • Registrasi Online
  • Jadwal Poli
  • Antrian Poli
  • Info Kamar
  • Kontak Aduan
  • Info Lainnya
  • KritikSaran
  • Profil Akun

Pendaftaran menggunakan Panti Nugroho Mobile bisa dilakukan dari H-7 sampai H-1 sebelum pemeriksaan, selain itu juga akan ditampilkan detail tanggal dan jam akan periksa, sehingga pasien tidak perlu menunggu lama.
